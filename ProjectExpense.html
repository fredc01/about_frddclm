Expense Management System

A centralized platform for submitting, approving, paying, and liquidating project expenses through multi-level processes.

Key Features

🖥️

Modern UI/UX

Crafted with Bootstrap and custom styles for a clean, responsive, and user-friendly interface.

🔄

Multi-Step Approval Workflow

Automated expense routing with confirm, approve, pay, and liquidate stages for transparency and accountability.

⏱️

Real-time Status Tracking

Users can monitor the status of their requests and approvers can act based on real-time data.

☁️

RESTful API Integration

Built with modular REST APIs for easy integration and clear separation between backend and frontend.

API Flow Examples

Submit New Expense

POST /api/expenses
{
  "title": "Hotel Booking",
  "amount": 1200,
  "submittedBy": "John Doe",
  "notes": "Business Trip to Tokyo",
  "receipts": ["receipt1.jpg", "receipt2.jpg"]
}

Confirm Expense

PUT /api/expenses/{id}/confirm
{
  "confirmedBy": "Finance Staff A"
}

Approve Expense

PUT /api/expenses/{id}/approve
{
  "approvedBy": "Manager X"
}

Mark Expense as Paid

PUT /api/expenses/{id}/pay
{
  "paidReference": "BANK-REF-987654",
  "paidAmount": 1200,
  "paidDate": "2025-04-28"
}

Submit Liquidation Result

PUT /api/expenses/{id}/liquidate
{
  "result": "Reimbursement",
  "remarks": "Additional taxi fare to be reimbursed."
}
