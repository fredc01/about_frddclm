Action Point Management System

A collaborative task resolution platform enabling teams to create, assign, track, and resolve project-related action points.

Key Features

📝

Create and Issue Action Point Cases

Create action point cases and assign contributors to ensure accountability and progress tracking.

👥

Assign Contributors and Target Dates

Easily assign contributors and set target dates to ensure timely completion of tasks.

✅

Add Tasks and Designate Persons in Charge

Break down action points into manageable tasks and designate responsible individuals for each task.

🔄

Resolve Individual Tasks or the Entire Case

Track and resolve individual tasks or the entire action point case for streamlined completion.

🔐

Role-Based Resolution Permissions

Enforce role-based access control for task resolution and case management.

API Flow Examples

Create Action Point Case

POST /api/action-cases
{
  "title": "Security Audit Checklist",
  "issuedBy": "Jane Lead",
  "targetDate": "2025-05-10",
  "contributors": ["John", "Sarah"]
}

Add Task to Case

POST /api/action-cases/{id}/tasks
{
  "description": "Update Firewall Rules",
  "personInCharge": "John Doe"
}

Resolve Single Task

PUT /api/action-cases/{id}/tasks/{taskId}/resolve
{
  "resolvedBy": "John Doe",
  "remarks": "Firewall rules updated"
}

Resolve All Tasks (by Issuer)

PUT /api/action-cases/{id}/resolve-all
{
  "resolvedBy": "Jane Lead"
}
